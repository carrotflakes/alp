# alp - example implementation of chat service

relies:

- Rust
- GraphQL (async-graphql)
- actix-web
- Next.js
- Firebase Authentication
- PostgreSQL

## Usage

```sh
$ docker-compose up
```

Then open localhost:3000 in your browser.

## Development

``` sh
$ cargo install diesel_cli --no-default-features --features postgres
```
